## 基础知识
- 并发编程的优缺点
  - 并发编程的优点:
    - 充分利用多核CPU的运算能力
    - 方便业务拆分，提升性能
  - 并发编程的缺点:
    - 频繁的上线文切换
    - 线程安全问题（常见的避免死锁的方式）
  
- 线程状态和基本操作
